Work in Progress. For now, the user is encouraged to try to match colors in the rainbow by selecting a color and finding the right slider combination to match it. A correct match will cause the color to appear in the cloud and for a random new color to be selected for matching (though the user may select a new color manually if they wish). The user may also interact with the cloud itself to see other angles.

If the user would like to see the full spectrum when complete, they may select "Next" and then go "Back". Currently, "Next" results in a WIP transition stage. By clicking "Next" again, the user can discover the cloud that this demonstration is working to complete, which is the cloud which contains most regularly defined display spaces (like sRGB or Display P3) using CIE 1931 RGB basis vectors. By further clicking “Next,” the viewer may see additional future stages, which include a chromaticity diagram derived from the CIE 1931 RGB color space, the standard chromaticity diagram, the previous cloud but using XYZ basis vectors, and a rotation of the previous states but with the Display P3 and sRGB spaces explicitly highlighted.

For best experience, please use desktop for the demonstration.
